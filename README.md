# 🐸 Frog Hero

**Frog Hero** is a story-driven **Rhythm Platformer** developed in **Unity** as my Senior Capstone Project. Combining platforming mechanics with rhythm-based gameplay, players guide a heroic frog through a musical adventure synchronized to a curated soundtrack.

As my first Unity project, Frog Hero served as an opportunity to learn game development from the ground up, including gameplay programming, level design, animation systems, audio synchronization, and cross-platform deployment.

## 🎮 Gameplay

* Navigate through platforming challenges as Frog Hero.
* Progress through a short narrative-driven adventure.
* Time movement and actions to the rhythm of the music.
* Experience levels designed around a custom soundtrack.
* Blend precision platforming with music-based mechanics.

## ✨ Features

## 🚀 Installation

1. Navigate to the **Builds** folder.
2. Download the ZIP file for your operating system.
3. Extract the ZIP file.

### Windows

Launch:

```text
Frog Hero/Rhythm Platformer.exe
```

### macOS

Launch:

```text
Frog Hero.app/Contents/MACOS/Rhythm Platformer
```

## ⚠️ Important

Keep all extracted game files in their original folder structure.

The executable depends on accompanying game assets and data files. Moving the executable outside of the extracted project directory may prevent the game from launching correctly.
